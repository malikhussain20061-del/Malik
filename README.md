# IVA SMS OTP Bot 🤖

Auto-forward OTPs from IVA SMS to Telegram Channel

## Features ✨
- 24/7 Auto-running (Serverless)
- Professional OTP formatting
- Clickable OTP codes
- Country & Service detection
- 1-second delay (safe rate limiting)

## Setup 🚀

### 1. Environment Variables
Set these in Vercel:
- `BOT_TOKEN` - Telegram Bot Token
- `CHANNEL_ID` - Telegram Channel ID
- `IVA_SESSION` - IVA SMS Session Cookie
- `IVA_CSRF` - IVA SMS CSRF Token
